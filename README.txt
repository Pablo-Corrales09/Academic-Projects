
<--------- English ---------->

Personal Goal Management System
This project is a web application designed to help users organize and manage their personal goals and projects more efficiently. 
It addresses the common problem of scattered notes and basic reminders, which make it difficult to prioritize tasks, track progress, and visualize long-term achievements.

Technologies used:
- C# with ASP.NET Core MVC.
- Entity Framework Core for data management.
 -SQL Server for database connectivity.

Features:
.Create, edit, and delete personal goals.
.Track progress and completion of tasks.
.Organize and prioritize objectives.
.Intuitive user interface for easy navigation.

This system is ideal for anyone looking to improve personal productivity and goal tracking through a centralized and structured platform.

<--------------------- Steps to Run the Project --------------------->

1. Open the Examen.sln file in Visual Studio.
2. Restore the NuGet packages.
Open the Package Manager Console and run the command: dotnet restore.
3. Apply the migrations to create the database.
Open the Package Manager Console and run the command: Update-Database.
4. You're all set! Run the project.

<--------- Español --------->

Sistema de Gestión de Metas Personales
Este proyecto es una aplicación web diseñada para ayudar a los usuarios a organizar y gestionar sus metas y proyectos personales de manera más eficiente. 
Soluciona el problema común de tener notas dispersas y recordatorios básicos, lo que dificulta priorizar tareas, hacer seguimiento del progreso y visualizar los logros a largo plazo.

Tecnologías utilizadas:
-C# con ASP.NET Core MVC
-Entity Framework Core para la gestión de datos.
-SQL Server para la conexión a la base de datos.

Funcionalidades:
-Crear, editar y eliminar metas personales.
-Hacer seguimiento del progreso y cumplimiento de tareas.
-Organizar y priorizar objetivos.
.Interfaz intuitiva para una navegación sencilla

Este sistema es ideal para cualquier persona que busque mejorar su productividad y seguimiento de metas a través de una plataforma centralizada y estructurada.


<--------------------- Pasos para ejecutar el proyecto --------------------->

1. Abre el archivo Examen.sln en Visual Studio.
2. Restaura los paquetes NuGet. 
- Abre Package Manager Console e ingresa el comando: dotnet restore.
3. Aplica las migraciones para que se cree la base de datos. 
- Abre Package Manager Console e ingresa el comando: Update-Database.
4. ¡Todo listo! Ejecuta el proyecto.
